# SolidWorks API
В этом приложении вы можете рисовать шестигранники, стороны которых являются дугами, и вырезать их из параллелепипеда с дополнительными условиями:
1) стороны соседних шестигранников не пересекаются.
2) дуги (стороны получившихся шестигранников) имеют разные радиусы.
3) фигуры не пересекают стороны параллелепипеда.
Также можно задавать шаг по оси Х и Y (расстояние между соседними шестигранниками), количество этих шестигранников.
